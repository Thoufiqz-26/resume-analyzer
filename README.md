# AI Resume Analyzer (Flask Web App)

A full-stack NLP-based ATS Resume Analyzer built using Flask and TF-IDF vectorization.

## Features
- Upload PDF Resume
- Paste Job Description
- Automatic Similarity Scoring
- Web-Based Interface (Bootstrap UI)
- Fully Offline (No API Required)

## Tech Stack
- Python
- Flask
- NLTK
- Scikit-learn
- PyPDF2

## How to Run
pip install -r requirements.txt
python app.py
